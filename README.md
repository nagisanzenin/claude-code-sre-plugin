# SRE Plugin for Claude Code

Site reliability -- production readiness, chaos engineering, capacity planning, and incident management.

## Installation

```
/plugin marketplace add nagisanzenin/claude-code-plugins
/plugin install sre@nagisanzenin
```

## Features

- Production readiness review
- Chaos engineering experiments
- Capacity planning and forecasting
- Incident management runbooks
- SLO/SLI/SLA definition
